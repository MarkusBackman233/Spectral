# Spectral
## Features
* DirectX11
* Lua script support
* Physically based rendering
## Dependencies
* [Assimp](https://github.com/assimp/assimp)
* [PhysX](https://github.com/NVIDIA-Omniverse/PhysX)
* [Rapidjson](https://github.com/Tencent/rapidjson/)
* [Lua](https://github.com/lua/lua/)
